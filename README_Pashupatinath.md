# 🐄 Pashupatinath - Livestock Health Intelligence System

## 📌 Project Overview
Pashupatinath is an AI-powered Android application designed for **early disease detection and outbreak prevention in cows and buffaloes**.  
The system works in **offline-first mode** and helps farmers identify diseases, predict herd-level risks, and take preventive actions.

---

## 🚀 Key Features
- 🧠 AI-based Disease Detection (Image + Symptoms)
- 📊 Herd Immunity Score Calculation
- ⚠️ Outbreak Risk Prediction Engine
- 📈 Contagion Spread Simulator
- 📱 Offline Edge AI Support
- 🩺 Action Recommendations (Isolation, Treatment, Prevention)
- 🌍 Farmer-friendly UI (Low literacy design)

---

## 🧩 Problem Statement
Livestock farmers face severe losses due to:
- Late disease detection
- Lack of veterinary access
- Rapid disease spread in herds

This project provides a **real-time intelligent solution** to:
- Detect diseases early
- Predict outbreaks
- Prevent large-scale losses
